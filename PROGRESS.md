# Love Message Website - Progress

## Project Overview
A simple, romantic interactive website where someone can create and share personalized love messages with beautiful animations and effects.

## MVP Scope
Single-page web app:
- Input recipient's name and custom message
- Generate a shareable page with romantic animations (hearts, sparkles)
- Mobile-responsive
- Can be hosted as static site (Azure SWA, GitHub Pages)

## Tech Stack
- Single HTML file (or minimal HTML/CSS/JS)
- No backend required (client-side only)
- CSS animations for effects
